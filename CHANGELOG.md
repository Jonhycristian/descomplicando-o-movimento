# Changelog

Todas as mudanças notáveis deste projeto serão documentadas neste arquivo.

O formato baseia-se em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e o projeto não segue estritamente o Versionamento Semântico no momento, por ser uma landing page estática.

## [Unreleased]

### Adicionado
- Melhorias profundas em SEO (Schema.org, Open Graph, Twitter Cards).
- Scripts automatizados para otimização de imagens (`loading="lazy"`).
- Atributos ARIA (`aria-label`) em botões não-textuais para conformidade com a WCAG.
- Geração de `sitemap.xml` e `robots.txt` para rastreamento dos motores de busca.
- README.md com documentação abrangente do projeto.
- Arquivos de repositório padrão: LICENSE, .gitignore e CONTRIBUTING.md.

### Alterado
- Refatoração do script de lazy loading para os vídeos do hero, melhorando a performance móvel.

## [1.0.0] - Lançamento Inicial
- Landing page principal estruturada com HTML Vanilla e CSS.
- Modais interativos com JS puro para vídeos e Typeform.
- Páginas satélites de "Consultoria", "Mentoria" e "Planilhas".
