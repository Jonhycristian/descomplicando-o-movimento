# Guia de Contribuição

Obrigado por se interessar em contribuir com o **Descomplicando o Movimento**!

## Como Contribuir

1. **Faça um Fork** do projeto.
2. **Crie uma branch** para sua feature/correção (`git checkout -b feature/nova-funcionalidade`).
3. **Commit** suas alterações (`git commit -m 'Adiciona nova funcionalidade XYZ'`).
4. **Push** para a branch (`git push origin feature/nova-funcionalidade`).
5. **Abra um Pull Request** detalhando as mudanças feitas.

## Padrões de Código

- Escreva HTML semântico.
- Siga as variáveis CSS definidas no arquivo `style.css`.
- Não adicione bibliotecas pesadas se o mesmo efeito puder ser atingido com Vanilla JS/CSS.
- Mantenha a acessibilidade (WCAG) em dia: adicione `alt` em imagens e `aria-label` em botões apenas com ícones.

## Reportando Bugs

Ao abrir uma Issue, por favor inclua:
- Comportamento esperado.
- Comportamento atual (bug).
- Print ou vídeo do problema.
- Dispositivo e navegador testado.
